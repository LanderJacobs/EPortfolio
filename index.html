Wie ben ik?

Ik ben Lander Jacobs momenteel student aan het Thomas More te Geel. Na het middelbaar was ik niet zeker wat ik wou doen en zo ben ik na verschillende keuzes beland bij het graduaat programmeren aan het Thomas More. Ik leerde veel bij over programmeren en de verschillende mogelijkheden naar de toekomst toe, dit lag mij wel. Zo wou ik na het behalen van mijn diploma voor dit graduaat graag verder gaan om mijn kennis te verbreden en een hoger diploma te halen door te gaan voor een bachelor in de Toegepaste Informatica.

Als ik een eigenschap van mezelf zou moeten benoemen zou ik zeggen dat ik vrij creatief uit de hoek kan komen, ik vindt het interessant om met creatieve oplossingen problemen uit de weg te slaan. Daarom heb ik ook gekozen voor applicatieprogrammeren als keuzerichting, dit is naar mijn mening een van de meest interessante onderwerpen binnen de IT-wereld om problemen te kunnen oplossen.

Project WebApplicatie

Een project voor het vak Webapplicaties binnen het graduaat programmeren. Het doel van de opdracht was om een Webapplicatie te maken in .net core met behulp van het MVC model om zo gegevens weer te geven vanuit een database die door de school werd onderhouden.

C#/.Net Core

Html

SQL

  • MVC (Model-View-Controller)
  • Dependency Injection
  • Code-first database
  • ORM (Object-relational mapping)
  • Unit of work
  • Generic repository

Binnen dit project heb ik voor de eerste keer gebruik gemaakt van MVC. Dit was een heel leerproces voor mij omdat dit mijn eerste aanraking met MVC/MVVM was. Daarnaast was het de bedoeling dat we hiervoor repositories zouden maken voor elke individuele klasse die in de database zat, ik heb deze kans gegrepen om mij meteen op pad te zetten met generic repositories. Dit maakte de rest van het programmeren wel een stuk eenvoudiger.
